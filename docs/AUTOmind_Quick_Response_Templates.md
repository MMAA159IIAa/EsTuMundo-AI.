# Manejo de Respuestas Rápidas (FAQ & Objections)

## Para Inversionistas
*   **Q: ¿Por qué ahora?**

*   **Q: ¿Cuál es el modelo de monetización?**
    *   *A:* Suscripción SaaS por taller + Tarifa por reporte generado + Comisión por venta de refacciones vía integraciones B2B.

## Para Talleres VIP
*   **Q: ¿Es compatible con todos los autos?**

*   **Q: ¿Necesito hardware especial?**
