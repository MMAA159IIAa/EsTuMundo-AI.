# Contributing to Es Tu Mundo AI

Thank you for your interest in contributing to **Es Tu Mundo AI**! 🚀

## 🏗️ Development Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/EsTuMundo-AI/estumundo-ecosystem.git
   cd estumundo-ecosystem
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env
   # Edit .env with your API keys
   ```

4. **Start development**
   ```bash
   npm run dev
   ```

## 📋 Guidelines

### Code Style
- Use **ES Modules** (`import`/`export`)
- Follow **ESLint** configuration
- Use **Prettier** for formatting
- Write meaningful commit messages

### Commit Convention
We follow [Conventional Commits](https://www.conventionalcommits.org/):

```
feat: add AI skincare recommendation engine
fix: resolve WhatsApp webhook timeout
docs: update Gemini integration guide
style: improve glassmorphism card effects
refactor: optimize agent pipeline processing
```

### Branch Strategy
- `main` — Production
- `develop` — Development  
- `feature/*` — New features
- `fix/*` — Bug fixes
- `docs/*` — Documentation

### Pull Requests
1. Create a feature branch from `develop`
2. Make your changes
3. Write/update tests
4. Submit a PR with a clear description
5. Wait for review

## 🤖 AI-First Development

All contributions should align with our AI-native philosophy:
- Leverage Gemini AI for intelligent features
- Automate repetitive workflows
- Build agents that solve real problems
- Optimize for user experience
